### 🌐 各市场接口详情

#### 1. A股实时行情

**接口地址**

```http
https://push2.eastmoney.com/api/qt/stock/get
```

**参数说明**

|参数|必须|说明|示例|
|---|---|---|---|
|`secid`|是|市场代码.股票代码|`0.000001`(深市平安银行)|
||||`1.600519`(沪市贵州茅台)|
|`fields`|是|需要返回的字段，用逗号分隔|`f57,f58,f2,f3,f4`|
|`fltt`|否|价格小数位长度|`2`|
|`invt`|否|不详，通常为`2`|`2`|

**市场代码 (`secid`前缀)**
- `0`: 深圳证券交易所
- `1`: 上海证券交易所
- （北交所代码通常为`0`，但建议通过数据列表接口确认）

#### 2. 港股实时行情

**接口地址**

```http
https://push2.eastmoney.com/api/qt/stock/get
```

**参数说明**

与A股接口相同，但 `secid`参数前缀不同。

|参数|必须|说明|示例|
|---|---|---|---|
|`secid`|是|市场代码.股票代码|`116.00700`(腾讯控股)|

**市场代码 (`secid`前缀)**

- `116`: 香港主板
    

#### 3. 美股实时行情

**接口地址**

```http
https://push2.eastmoney.com/api/qt/stock/get
```

**参数说明**

与A股接口相同，但 `secid`参数前缀不同。

|参数|必须|说明|示例|
|---|---|---|---|
|`secid`|是|市场代码.股票代码|`105.AAPL`(苹果)|

**市场代码 (`secid`前缀)**

- `105`: 美股市场
    

#### 4. 场外基金净值

**接口地址**

```http
http://fund.eastmoney.com/f10/F10DataApi.aspx
```

**参数说明**

|参数|必须|说明|示例|
|---|---|---|---|
|`type`|是|数据类型，固定为`lsjz`|`lsjz`|
|`code`|是|基金代码|`000001`|
|`page`|是|页码|`1`|
|`sdate`|是|开始日期 (`YYYY-MM-DD`)|`2023-01-01`|
|`edate`|是|结束日期 (`YYYY-MM-DD`)|`2023-12-31`|

#### 5. K线数据 (A/H/美股、指数、ETF通用)

**接口地址**

```http
https://push2his.eastmoney.com/api/qt/stock/kline/get
```

**参数说明**

|参数|必须|说明|示例|
|---|---|---|---|
|`secid`|是|市场代码.股票代码|`1.000300`(沪深300指数)|
|`klt`|是|K线周期|`101`(日线)|
||||`102`(周线)|
||||`1`(1分钟)|
||||`5`(5分钟)|
|`fqt`|是|复权类型|`0`(不复权)|
||||`1`(前复权)|
||||`2`(后复权)|
|`lmt`|否|数据条数|`100`|
|`beg`/ `end`|否|开始/结束日期|`20230101`|

#### 6. 资金流向 (个股)

**接口地址**

```http
https://push2.eastmoney.com/api/qt/stock/fflow/kline/get
```

**参数说明**

此接口参数与K线接口类似，返回包含主力、散户等资金流向数据。`klt`参数通常设为 `101`(日线)。

### 💻 调用示例

**1. 获取贵州茅台实时行情**

```http
https://push2.eastmoney.com/api/qt/stock/get?secid=1.600519&fields=f57,f58,f2,f3,f4,f5
```

返回数据将包含代码、名称、最新价、涨跌幅、涨跌额、成交量。

**2. 获取沪深300指数近100条日K线数据**

```http
https://push2his.eastmoney.com/api/qt/stock/kline/get?secid=1.000300&klt=101&fqt=1&lmt=100
```

### ⚠️ 注意事项

1. **数据延迟**：接口获取的数据可能存在15分钟或更长时间的延迟，不适用于实时交易决策。
    
2. **字段变动**：字段标识（如`f2`, `f3`）及其含义可能随东方财富网站更新而变动，需要定期验证。
    
3. **商业用途**：如需将数据用于商业项目，请务必查阅东方财富的服务协议，或考虑使用其官方付费数据服务（如Choice数据）。
