# 策略回测功能使用指南

## 📊 功能概述

策略回测功能允许您测试不同的交易策略在历史数据上的表现，帮助您优化投资决策。

---

## ✨ 主要功能

### 1. **支持的策略类型**

- **移动平均线策略 (MA)** - 基于短期和长期均线交叉
- **MACD 策略** - 基于MACD指标的金叉死叉
- **KDJ 策略** - 基于KDJ指标的超买超卖
- **RSI 策略** - 基于相对强弱指数
- **布林带策略** - 基于价格突破布林带

### 2. **回测配置**

- 股票代码选择
- 回测时间范围（开始日期-结束日期）
- 初始资金设置
- 手续费率配置
- 策略参数自定义

### 3. **结果展示**

#### **关键绩效指标**
- **总收益率** - 整个回测期间的总收益
- **年化收益率** - 换算成年化后的收益率
- **最大回撤** - 从峰值到谷底的最大跌幅
- **夏普比率** - 风险调整后的收益指标
- **胜率** - 盈利交易占总交易的比例
- **交易次数** - 完整买卖对数

#### **可视化图表**
- **收益曲线图** - 展示资产净值随时间的变化
- **交易记录表** - 详细的每笔交易记录

---

## 🎯 使用步骤

### 第1步：配置回测参数

1. **输入股票代码**
   - 例如：`600519`（贵州茅台）
   - 例如：`000001`（平安银行）

2. **选择策略类型**
   - 从下拉菜单中选择您想测试的策略

3. **设置时间范围**
   - 开始日期：回测的起始日期
   - 结束日期：回测的结束日期
   - 默认：最近一年

4. **配置资金参数**
   - 初始资金：默认 100,000 元
   - 手续费率：默认 0.03%

5. **调整策略参数**
   - 根据选择的策略类型，参数会动态变化
   - 例如MA策略：短期均线（默认5日）、长期均线（默认20日）

### 第2步：运行回测

点击 **"开始回测"** 按钮

- 系统将生成历史数据（当前使用模拟数据）
- 执行策略逻辑，计算买卖信号
- 模拟交易过程，记录每笔交易

### 第3步：查看结果

回测完成后，页面会自动滚动到结果区域，显示：

1. **绩效指标卡片** - 6个关键指标一目了然
2. **收益曲线图** - 可视化资产变化趋势
3. **交易记录表** - 详细的交易明细

## 📈 策略详解

### 1. 移动平均线策略 (MA)

**原理：**
- 短期均线上穿长期均线（金叉）→ 买入信号
- 短期均线下穿长期均线（死叉）→ 卖出信号

**参数：**
- `短期均线` - 建议：5-10日
- `长期均线` - 建议：20-60日

**适用场景：** 趋势明显的市场

---

### 2. MACD 策略

**原理：**
- MACD线上穿信号线 → 买入
- MACD线下穿信号线 → 卖出

**参数：**
- `快线周期` - 默认：12
- `慢线周期` - 默认：26
- `信号线周期` - 默认：9

**适用场景：** 中长期趋势跟踪

---

### 3. KDJ 策略

**原理：**
- K值 < 超卖线（如20）→ 买入
- K值 > 超买线（如80）→ 卖出

**参数：**
- `K值周期` - 默认：9
- `超买线` - 默认：80
- `超卖线` - 默认：20

**适用场景：** 震荡市场，短期交易

---

### 4. RSI 策略

**原理：**
- RSI < 超卖线 → 买入
- RSI > 超买线 → 卖出

**参数：**
- `RSI周期` - 默认：14
- `超买线` - 默认：70
- `超卖线` - 默认：30

**适用场景：** 判断超买超卖状态

---

### 5. 布林带策略

**原理：**
- 价格跌破下轨 → 买入
- 价格突破上轨 → 卖出

**参数：**
- `均线周期` - 默认：20
- `标准差倍数` - 默认：2

**适用场景：** 波动率交易

---

## 💡 使用技巧

### 1. **参数优化**

- 不要过度优化（避免过拟合）
- 测试不同时间周期
- 考虑不同市场环境

### 2. **风险管理**

- 关注最大回撤指标
- 夏普比率 > 1 表示风险调整后收益较好
- 控制单次交易仓位

### 3. **策略组合**

- 可以测试多个策略
- 比较不同策略的表现
- 考虑策略的互补性

### 4. **注意事项**

⚠️ **重要提示：**
- 回测结果不代表未来收益
- 当前使用模拟数据，实际应用需接入真实数据
- 交易成本会影响收益，需考虑手续费、滑点等
- 历史表现不能保证未来表现

## 📊 性能指标说明

### 总收益率
```
总收益率 = (期末资产 - 期初资产) / 期初资产 × 100%
```

### 年化收益率
```
年化收益率 = (期末资产 / 期初资产)^(1/年数) - 1
```

### 最大回撤
```
最大回撤 = max((峰值 - 当前值) / 峰值)
```

### 夏普比率
```
夏普比率 = (平均收益 - 无风险利率) / 收益标准差
```
- 值越高越好
- > 1：较好
- > 2：优秀
- > 3：卓越

### 胜率
```
胜率 = 盈利交易次数 / 总交易次数 × 100%
```
