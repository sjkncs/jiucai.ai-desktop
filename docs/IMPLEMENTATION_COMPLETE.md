## 📞 测试反馈

如果遇到问题，请检查：

1. **个股分析不显示**
   - 确认输入的是 600519 或 000001
   - 检查开发者工具Console是否有错误
   - 确认stock.js正确加载

2. **投资组合数据丢失**
   - 检查浏览器LocalStorage是否被清除
   - 确认表单所有字段都已填写
   - 查看Console是否有保存错误

3. **K线图不显示**
   - 等待1-2秒让Canvas加载
   - 刷新页面重试
   - 检查Canvas元素是否存在
