**欢迎使用久财AI桌面应用！**

## ⚡ 立即启动

### 方式1: 一键启动（推荐）

双击运行以下任一文件：

```
START_ALL.bat          # 启动所有服务（Desktop + Website + API）
desktop-app\START_APP.bat  # 仅启动桌面应用
```

### 方式2: 命令行启动

```bash
# Desktop App
cd desktop-app
npm start

# Website (新终端)
cd website  
npm run dev
```
